+++оси данных для рекомендаций
- категории
- теги
- термы (подготовленные слова из текстов, описаний, названий)
- авторы

каждый курс имеет
- автора,
- категории,
- теги
- термы

пользователь имеет
- гео
- пол
- возраст

при просмотре курсов пользователь обрастает интересами
интересы накапливаются и меняются со временем
имеет смысл работать с временными окнами интересов
размер окна - 30дней
можно искать похожие окна от разных пользователей, причём окна могут быть из разных врменных промежутков, т.е. А в этом месяце похож на Б год назад (контент устаревает - имеет ли это смысл?)



+++организация окон

данные окна по пользователю
- гео
- пол
- возраст
- количество просмотров по
    - категориям
    - тегам
    - термам
- курсы
- количество курсов


данные по регионам
- гео
- пол
- возраст
- топ100 курсов
- топ100 курсов по
    - категориям

+++Проблема холодного старта в начале каждого месяца
пути решения
- скользящие окна
- сумма окон с затуханием (со временем вес исторических данных затухает)

вариант суммы с затуханием более предпочтительный в плане простоты организации

+++поиск рекомендаций пользователю по окну интересов
идея - найти интересуемый контент по окну
- берём окно пользователя
- делаем 3 оси распределений тэги категории термы
- строим запрос по курсам с бустом по осям распределений

+++поиск рекомендаций пользователю по похожим пользователям
идея - найти интересуемый контент у похожих по интересам пользователей с более богатой историей просмотров (есть ли смысл применить подобие вместо абсолютных чисел интересов?)
- берём окно
- делаем 3 оси распределений
- по ним ищем топ 20 похожих пользователей
- у пользователей смотрим курсы

+++поиск рекомендаций пользователю в случае холодного старта пользователя
- берём гео пол возраст и ищем окно
- в окне берём топ100 или топ100 по категориям из профиля

+++поиск рекомендаций пользователю в случае холодного старта системы
- формируется лента курсов в порядке добавления
- можно формировать ленты по категориям
